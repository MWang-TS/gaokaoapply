# PDF 导入功能说明

## 功能概述

本项目已添加从 PDF 文件导入高考分数线数据的功能。支持导入上海市普通高校招生本科普通批次平行志愿院校专业组投档分数线 PDF 文件。

## 使用方法

### 1. 构建与运行

```bash
# 进入项目目录
cd gaokaoapply

# 安装依赖
npm install

# 开发模式运行
npm run tauri dev

# 或构建生产版本
npm run tauri build
```

### 2. 导入 PDF 数据

1. 点击左侧导航栏的 **"📊 高考数据"** 标签
2. 在筛选栏右侧点击 **"📄 导入 PDF"** 按钮
3. 在弹出的文件选择对话框中选择 PDF 文件（例如：`上海市2025年普通高校招生本科普通批次平行志愿院校专业组投档分数线.pdf`）
4. 等待 PDF 解析完成（几秒钟）
5. 在预览窗口中确认解析到的数据：
   - 年份和省份可以手动修改
   - 表格显示前 50 条记录，包含院校代码、名称、投档线
6. 点击 **"✅ 确认导入"** 保存数据
7. 页面会自动刷新，新数据会显示在数据视图中

### 3. 数据格式

导入后数据将保存为 `shanghai_2025_cutoffs.json`，格式如下：

```json
{
  "title": "2025年上海普通高校招生本科普通批次平行志愿院校专业组投档分数线",
  "year": 2025,
  "province": "上海",
  "note": "由 PDF 导入生成",
  "data": [
    { "code": "10101", "name": "复旦大学(01)", "cutoff": 580 },
    { "code": "10102", "name": "复旦大学(02)", "cutoff": "580分及以上" },
    ...
  ]
}
```

## 技术实现

### 后端 (Rust)

- **PDF 解析库**: `pdf-extract` - 提取 PDF 文本内容
- **文本处理**: `regex` - 正则表达式匹配院校代码、名称、分数
- **编码处理**: `encoding_rs` (GB18030) - 处理中文 PDF 编码

关键文件：`src-tauri/src/lib.rs`
- `import_pdf_cutoffs` - PDF 解析命令
- `open_file_dialog` - 文件选择对话框
- `parse_cutoff_lines` - 文本行解析逻辑

### 前端 (React/TypeScript)

- 使用 Tauri `invoke` 调用 Rust 命令
- 预览模态框显示解析结果
- 确认后通过 `save_gaokao_data` 保存

关键文件：`src/App.tsx` (DataView 组件)
- `handleImportPdf()` - 触发 PDF 导入
- `handleConfirmImport()` - 确认并保存
- 导入预览 UI

## 支持的 PDF 格式

当前解析器支持以下行格式：

```
10101 复旦大学(01) 580
10102 复旦大学(02) 580分及以上
10301 同济大学(01) 560
```

- 第一列：5位及以上数字（院校代码）
- 中间列：非数字字符（院校专业组名称）
- 最后一列：数字或"数字分及以上"（投档线）
- 支持多列表格中夹杂选考要求等额外列的情况

## 注意事项

- PDF 文件需要包含清晰的表格文本
- 扫描版 PDF（图片）无法解析，需要 OCR
- 导入前会先预览，确认无误后再保存
- 原始数据会被覆盖，建议先备份
